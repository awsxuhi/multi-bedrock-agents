# 架构图占位符

这个目录将包含项目的架构图和其他资源文件。

## 架构图

架构图应该展示以下组件之间的关系：

1. 用户/客户端
2. Streamlit 前端
3. Supervisor Agent
4. Portfolio Creator Agent
5. Knowledge Base
6. OpenSearch Serverless
7. S3 存储桶
8. Lambda 函数

## 示例架构图

```
+----------------+     +-------------------+     +---------------------+
|                |     |                   |     |                     |
|  用户/客户端    +---->+  Streamlit前端    +---->+  Supervisor Agent   |
|                |     |                   |     |                     |
+----------------+     +-------------------+     +---------+-----------+
                                                          |
                                                          v
                                                +---------+-----------+
                                                |                     |
                                                | Portfolio Creator   |
                                                | Agent               |
                                                |                     |
                                                +---------+-----------+
                                                          |
                              +------------------------+  |  +------------------------+
                              |                        |  |  |                        |
                              |  Knowledge Base        <--+->+  Lambda函数            |
                              |                        |     |                        |
                              +------+----------+------+     +------------------------+
                                     |          |
                                     v          v
                       +-------------+----+    ++-----------------+
                       |                  |    |                  |
                       | OpenSearch       |    | S3存储桶         |
                       | Serverless       |    |                  |
                       |                  |    |                  |
                       +------------------+    +------------------+
```

## 注意

这只是一个简单的占位符，实际的架构图应该使用专业的绘图工具创建，如 Draw.io、Lucidchart 或 Visio。
